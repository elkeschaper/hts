Tutorial: Write data¶

See how to install HTS.

Load all dependencies and load your data¶

[1]:

# %matplotlib inline is only necessary in inline mode
%matplotlib inline
import logging
import os

from hts.data_tasks import qc_matplotlib
from hts.run.run import Run
from hts.paths import TUTORIAL_DATA_NORMALIZATION_PATH

logging.basicConfig(level=logging.ERROR)
config_path = TUTORIAL_DATA_NORMALIZATION_PATH

run = Run.create(path=config_path, origin="config")

Write your data¶

A couple of standard formats are currently available:

[2]:

run.write(format="csv", path="~/run.csv")
run.write(format="csv_one_well_per_row", path="~/run_common.csv")
run.write(format="pickle", path="~/my_run.pickle")

You can also access the data as a pandas data.frame:

[ ]:

run.write(format="serialize_as_pandas")

It is also possible to write the data via pandas:

[ ]:

data_frame = my_run.data_frame
data_frame_sample_only = my_run.data_frame_samples
data_frame_sample_only.to_csv("~/run_from_pandas.csv") # Add Pandas parameters for filtering (e.g. columns=["a", "b"]).
