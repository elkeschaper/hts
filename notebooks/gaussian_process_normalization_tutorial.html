2. Read in your high throughput screening data¶

Let’s read in some data defined in a config file:

In [11]:

Find the configuration file for this tutorial here: /Users/elkeschaper/Python_projects/HTS/test_data/Runs/run_config_tutorial_1.txt

We create a Run instance from the original raw data. A Run can hold all information on readouts, plate layouts, and meta data on the samples. A Run instance provides all functionality to deal with the data, such as for input/output, quality control and analysis reporting, and hit selection. The RBF Kernel¶

In our Protocol config, we see the following task:

[Tutorial data: Train a Gaussian process on sample data]
tags = analysis, gp  # Choose arbitrary tags for your task
method = gaussian_processes.add_gaussian_processes
data_tag_readout = net_fret # What readout should we train the GP on?
sample_tags = s # Which wells should we train the GP on [s: sample wells, neg: negative control, pos: positive control, ...]
[[rbf]] # Arbitrarily name your GP.
    is_per_plate = True # Train one GP per plate.
    is_zigzag = False # ToDo: Remove.
    [[[RBF]]]
        kernel_type=RBF # Choose the GPy kernel
        [[[[constraints]]]] # Choose arbitrary constraints on the GPy kernel parameters.
            [[[[[lengthscale]]]]]
                fix = 3
            [[[[[variance]]]]]
                fix = 1

Here, for each plate, we added a Gaussian process named “rbf”. We defined what data the Gaussian process should be trained on: sample_tags = 2 For every Gaussian process, we can define an arbitrarily constituted kernel. In this case, we chose only a very simple kernel: The RBF kernel. All kernels implemented in GPy for at least two-dimensional data can be used. Look at the Overview of all GPy kernels

Further down in Protocol config, we see this task:

[Tutorial data: Use the trained Gaussian process for prediction/smoothing.]
    tags = analysis, gp
    method = gaussian_processes.do_gaussian_process_prediction
    data_tag_readout = net_fret # ToDo: Is this needed?
    best_mean_tag = best__s__predicted_mean
    best_sd_tag = best__s__predicted_sd
    sample_tags = s, # ToDo: What is this good for?
