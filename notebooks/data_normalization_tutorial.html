Tutorial: Data normalization¶

Read in your high throughput screening data¶

First, we read in a simple example of high throughput screening data. We prepared example data for you. For reading in your own data, there are detailed info on Data IO.

In [1]:

%matplotlib inline
import os

from hts.run.run import Run
from hts.paths import TUTORIAL_DATA_NORMALIZATION_PATH


data_path = TUTORIAL_DATA_NORMALIZATION_PATH
print("If you would like to see the data file, go to {}".format(data_path))

We create a Run instance from the original raw data. A Run can hold all information on readouts, plate layouts, and meta data on the samples. A Run instance provides all functionality to deal with the data, such as for input/output, quality control and analysis reporting, and hit selection. For more information on Run instances, and create statements, look here Data IO and link to RUN. [TODO: In Run, add schema of Run/Plate/PlateData]
